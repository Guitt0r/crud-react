# CRUD-app
This project aims to create a versatile CRUD (Create, Read, Update, Delete) application using React
## Technologies
- React.js
- Redux Toolkit
- Typescript
- TailwindCss
- React Hook Form

## Installation
Use [yarn](https://yarnpkg.com/) as your package manager to install dependencies:
```
yarn
```
## Run
```
yarn dev
```
